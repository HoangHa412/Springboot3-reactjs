# Full Stack Employee Management Web Using ReactJs and Springboot

# Tech We Used
- ReactJs
- SpringBoot
- MySql Database
- Spring Security
- REST API
- Oauth2
- Websocket

# Features
- Login, Login with Google, Logout, Forgot password and Changed password
- CRUD Operations
- Data are stored in Mysql database and
- Neat and clean UI
